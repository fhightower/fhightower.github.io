- [TCEX App Template](https://github.com/fhightower-templates/tcex-app-template): Template for creating a quality TCEX app (either a playbook app or python job app) quickly (once everything is installed, it take about 30 seconds to get a working app).
- [ThreatConnect Python SDK Snippets](https://packagecontrol.io/packages/ThreatConnect%20Python%20Snippets): Sublime Text snippets that let you write scripts using the ThreatConnect Python SDK less than 60 seconds.
- [Playbook Paradigms and Constructs](https://fhightower.gitbooks.io/threatconnect-playbook-paradigms-and-constructs/content/): Tips and tricks for building quality and maintainable playbooks.
- [Keyboard Shortcuts for ThreatConnect](https://github.com/fhightower/threatconnect-keyboard-shortcuts): Enough said. A joy to use!
- [Playbook Utility](http://playbooks.hightower.space): Helpful tools for creating and working with playbooks. Some notable tools are:
  - [HTTP Request Playbook Creator](http://playbooks.hightower.space/requester): System for quickly and painlessly creating a playbook which makes a request to a website/endpoint (providing the site returns json) and selects specific paths from the response json.
  - [Open Source Playbook Explorer](http://playbooks.hightower.space/explore): Explore the playbooks, components, and apps listed on [https://github.com/ThreatConnect-Inc/threatconnect-playbooks](https://github.com/ThreatConnect-Inc/threatconnect-playbooks).
- [IOC Toolkit](http://iocs.hightower.space/): This isn't directly related to ThreatConnect, but this site contains some helpful resources for working with Indicators of Compromise. What is especially nice about this site is that it contains API endpoints for all of the functions it can perform which makes it possible to use this system from a playbook.
- [UI for Creating TCEX Apps](http://tcex.hightower.space/): This is coming soon (April 2018)!
- [ThreatConnect Populator](https://github.com/fhightower/threatconnect-populator): Script to create test data in ThreatConnect... helpful for testing and general messing around. It creates one of every possible item in TC.
