ThreatConnect Resources

Projects related to ThreatConnect.

Disclaimer: None of the resources listed below are managed by ThreatConnect. Almost all (if not all) of the projects below are licensed under the MIT license. You are encouraged to use the resources below, but realize that the author is not responsible for any damages caused by these resources or the use of these resources. Enjoy!

  • TCEX App Template: Template for creating a quality TCEX app (either a playbook app or python job app) quickly (once everything is installed, it take about 30 seconds to get a working app).
  • ThreatConnect Python SDK Snippets: Sublime Text snippets that let you write scripts using the ThreatConnect Python SDK less than 60 seconds.
  • Playbook Paradigms and Constructs: Tips and tricks for building quality and maintainable playbooks.
  • Keyboard Shortcuts for ThreatConnect: Enough said. A joy to use!
  • Playbook Utility: Helpful tools for creating and working with playbooks. Some notable tools are:
    • HTTP Request Playbook Creator: System for quickly and painlessly creating a playbook which makes a request to a website/endpoint (providing the site returns json) and selects specific paths from the response json.
    • Open Source Playbook Explorer: Explore the playbooks, components, and apps listed on https://github.com/ThreatConnect-Inc/threatconnect-playbooks.
  • IOC Toolkit: This isn’t directly related to ThreatConnect, but this site contains some helpful resources for working with Indicators of Compromise. What is especially nice about this site is that it contains API endpoints for all of the functions it can perform which makes it possible to use this system from a playbook.
  • UI for Creating TCEX Apps: This is coming soon (April 2018)!
  • ThreatConnect Populator: Script to create test data in ThreatConnect… helpful for testing and general messing around. It creates one of every possible item in TC.

For more information about this project, refer to: projects/threatconnect/.
